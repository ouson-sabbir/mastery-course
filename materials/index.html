Three.js Materials Learning Hub

Each material now has its own folder, its own focused example, and its own separate note file so you can teach one topic at a time without a giant code file.

Recommended class flow: open one lesson, explain the note, show the small example, then finish with the final showroom for comparison.

MeshBasicMaterialNo light reaction. Best for simple objects and wireframe. MeshLambertMaterialSimple matte lighting without shiny highlights. MeshPhongMaterialClassic shiny highlights and specular control. MeshStandardMaterialRealistic PBR material with roughness and metalness. MeshPhysicalMaterialAdvanced realistic glass and premium surfaces. MeshToonMaterialStylized cartoon-like shading. MeshNormalMaterialShows normal direction using color. MeshMatcapMaterialFake baked lighting using a matcap texture. MeshDepthMaterialVisualize depth from the camera. MeshDistanceMaterialDistance-based rendering for advanced effects. ShadowMaterialInvisible receiver that only shows shadows. LineBasicMaterialSolid color line rendering. LineDashedMaterialDashed line rendering with dash size and gap size. PointsMaterialParticles and point cloud rendering. SpriteMaterialCamera-facing label or icon material. ShaderMaterialCustom GLSL with Three.js support. RawShaderMaterialFull raw shader control for advanced lessons.

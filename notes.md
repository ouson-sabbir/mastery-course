# Notes

- This workspace is a simple Three.js starter for learning the basics.
- Open [basic/index.html](./basic/index.html) with a local server like VS Code Live Server for the smoothest experience.
- `index.html` sets up the page and loads Three.js from a CDN.
- `script.js` creates the scene, camera, renderer, cube, helpers, lights, and animation loop.

## Beginner Experiments

- Change the cube color in `MeshStandardMaterial`.
- Move the camera closer or farther away.
- Replace `BoxGeometry` with `SphereGeometry`.
- Comment out the helpers to see a cleaner scene.
- Slow down or speed up the cube rotation.
